# 🧠 Deepfake Detection App (Image & Video)

A simple AI-powered web application to detect potential deepfake images and videos using basic computer vision techniques.

Built with:
- Streamlit (UI)
- OpenCV (image/video processing)
- NumPy (data handling)
- PIL (image processing)

---

## 🚀 Features

- 📷 Image Deepfake Detection
- 🎥 Video Deepfake Detection
- ⚡ Fast and lightweight
- 🌐 Easy-to-use web interface

---

## ⚠️ Important Note

This project uses **basic heuristic methods**, NOT a trained deep learning model.

- Image detection is based on **Laplacian variance (blur detection)**
- Video detection is based on **frame count analysis**
